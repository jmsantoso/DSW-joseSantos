Hojas de estilo en cascada

¿Qué es CSS?



CSS se refiere a las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets), Se denomina «en cascada» porque las propiedades del diseño pueden ser heredadas (o «en cascada») de otras. funciona como bloques de instrucciones anidadas.

Resulta útil porque optimiza el funcionamiento de las páginas web al estar separado del contenido, esto hace que el procesamiento de la información sea más rápido, esto tiene como ventaja una experiencia más fluida.

Se trata de un lenguaje que gobierna el diseño y presentación de las páginas web, con el puedes personalizar y ajustar la apariencia de un sitio web, su aspecto visual, el cómo presentaras la información variando las fuentes, los colores los tamaños y demás aspectos, todo esto separado del contenido, así podrás destacar la página en general o los aspectos que consideres relevantes.

Es importante destacar que CSS funciona en conjunto con el lenguaje HTML que se encarga del contenido básico de las páginas. (“Introducción al CSS: qué es, para qué sirve y otras 10 preguntas frecuentes”).

CSS es muy útil porque sin importar el navegador te permitirá obtener siempre los mismos resultados, esto se extiende a un amplio número de dispositivos

Frameworks CSS



Un framework es un conjunto de herramientas, compuesto por librerías, convenciones y buenas prácticas que pretenden encapsular las tareas repetitivas en módulos genéricos fácilmente reutilizables. (“Capítulo 5. Frameworks (CSS avanzado) - uniwebsidad.com”) En el caso específico de frameworks CSS son conjuntos de herramientas, hojas de estilo y buenas prácticas que ayudan con las tareas repetitivas permitiéndonos centrarnos en elementos únicos que puedan aportar valor al diseño web.

Es habitual en frameworks CSS que se incluyan herramientas varias, las más usuales son:
  • Para neutralizar los estilos que aplican los navegadores por defecto
  • Que el texto sea adaptable para mejorar su accesibilidad en cualquier medio o dispositivo
  • Manejar correctamente el texto, logrando que todos los contenidos se vean exactamente igual en todos los navegadores (“Capítulo 5. Frameworks (CSS avanzado) - uniwebsidad.com”)
  • Crear estructuras complejas de forma sencilla, con la seguridad de que funciona correctamente en cualquier versión de cualquier navegador.

Cuando apareció creó una tendencia de frameworks basados en componentes, capaces de implementar temas de diseño completos y complejos, aportando mucha sencillez y agilidad al desarrollo CSS y dotando a los programadores de herramientas para crear diseños consistentes con poco esfuerzo.     Materialize CSS
    Materialize es un framework CSS que implementa el tema de diseño "Material Design" que es un sistema adaptable de pautas, componentes y herramientas que soportan las buenas prácticas en diseño de interfaces, fue desarrollado por Google. Ofrece componentes material listos para usar, que se pueden integrar de una manera cómoda en los sitios web, consiguiendo un diseño guiado por las directrices de aplicaciones y sitios de Google. Crea diseños consistentes que son fáciles de entender y familiares para los usuarios en general. Tiene muchos componentes que también implementan comportamientos dinámicos con JavaScript y recientemente se actualizó para eliminar su dependencia con jQuery. (“Frameworks CSS - Desarrollo Web”) El framework es sencillo de usar, relativamente ligero, permite optimización y los componentes están altamente personalizados en su diseño. https://github.com/Dogfalo/materialize
    Tailwind CSS
    Este es un framework CSS que hace parte de los llamados "utility first" ofreciendo estilos CSS atómicos. Es un enfoque diferente, en realidad no tiene muchos componentes, sino clases de utilidad que aplicar directamente sobre el CSS. Aunque también permite crear componentes, lo deja más del lado del desarrollador para los personalice a su gusto. Tailwind CSS tiene la característica de ser maleable y adaptarse a las necesidades del desarrollador. Permite hacer construcción de clases CSS totalmente personalizadas, también cuenta con clases predeterminadas. (“Frameworks CSS - Desarrollo Web”) https://github.com/tailwindlabs/tailwindcss
    Foundation
    Es uno de los frameworks CSS basados en componentes más importantes del momento. A diferencia de otros, ofrece un subconjunto de sus herramientas orientado a la creación de emails, que tienen características de desarrollo distintos que los sitios web. Ofrece un set de herramientas para poder optimizarlo correctamente y construir un sitio que incluya solamente los componentes que se desean. (“Frameworks CSS - Desarrollo Web”) https://github.com/foundation/foundation-sites
    Bulma
    Framework CSS basado en componentes que usa Flexbox de manera precisa para simplicidad del código. Los componentes de Bulma son CSS puros, es decir, no tiene el set de componentes que requieren JavaScript para funcionar. Pero puede incorporar extensiones que sí incluyen componentes de todo tipo, dinámicos y con JavaScript, usando varios frameworks, creados por terceros. Es sencillo de usar y de aprender y permite un desarrollo moderno. https://github.com/jgthms/bulma
    Tachyons
    Es un framework CSS basado en clases de utilidad, con CSS atómico. El mismo enfoque que Tailwind CSS con diferencia que este está pensado para ser ligero y sencillo, con un CSS poco pesado (aproximadamente 14KB) que se puede usar sin necesidad de herramientas de optimización. (“Frameworks CSS - Desarrollo Web”) https://github.com/tachyons-css/tachyons/
